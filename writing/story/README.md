# Story — Survive the 70s

This folder is the **working draft of the playable spine**, organized act-by-act.

## Purpose

A middle layer between the long-form session notes
(`writing/sessions/2026-05-24_tear-money-and-handoff.md`) and the actual
Ren'Py script (`game/script.rpy`). Use this folder to:

1. Read the story cleanly when planning the next Ren'Py scene.
2. Edit/refine beats without diving into the giant session file.
3. Mark beats as **LOCKED** when they're ready to move to script.

## Files

| File | Covers |
|---|---|
| `00-opening.md` | Cold open: saturated 2027 → switch → wake-up as Danny → smoke detector → READ ME letter. Ends at title card. |
| `01-act1.md` | Tutorial morning + detective-game Act 1. Begins with TV/Bobby/Kay scene. Ends when comms with Danny opens. |
| `02-act2.md` | Comms open. Scavenge/craft/Iwona arc. Ends with Danny's *"I'm getting the idea you don't want to come back."* |
| `03-act3.md` | Oceanside. Amber. Bicentennial concert. Kiss. Swap fires. |
| `04-ending.md` | Mark returns to 2027. Danny old, sick of it, sequel hook. |

## Status legend

Each beat is marked:

- 🔒 **LOCKED** — ready to script in Ren'Py. Do not change without lock-break note.
- 📝 **DRAFT** — captured but still movable.
- ❓ **OPEN** — known gap, needs design pass.

## Authoritative sources

- **Canon files** (worldbuilding-rules.md, decisions-log.md, etc.) override anything here.
- **This folder** overrides `story-arc.draft.md` and `scene-outlines.md` (both archive).
- **Session file** is historical record — quote from it, don't link players to it.
